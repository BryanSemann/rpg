# Fase 0 - Checklist de Pronto

Date: 2026-03-14
Scope: definicao de guardrails de produto e compliance antes das fases de implementacao.

## Objetivo

Fechar oficialmente as decisoes de escopo, risco e governanca do MVP.

## Checklist

- [x] Definir escopo de conteudo do MVP: SRD/OGL only.
- [x] Definir plataforma inicial: Android-first.
- [x] Definir estrategia de co-criacao de UI: wireframe textual antes de codar.
- [x] Publicar ADR de politica de conteudo.
- [x] Publicar politica de fontes externas aceitas.
- [x] Definir gate de compliance para entrada de nova fonte.
- [x] Registrar matriz comparativa final de fontes (DnD5eAPI vs Open5e) com decisao primaria/fallback.
- [x] Definir metricas minimas da Fase 1 (cobertura de entidades MVP por fonte).

## Definition of Done (Fase 0)

A Fase 0 sera considerada concluida quando:

1. Todos os itens marcados como obrigatorios estiverem completos.
2. A fonte primaria e a fonte de fallback forem aprovadas com justificativa.
3. O time concordar que nenhuma feature da Fase 1 viola o ADR 0001.

## Risco Residual

- Cobertura parcial do ecossistema D&D 5e no MVP.
- Necessidade de normalizacao de dados entre fontes.
